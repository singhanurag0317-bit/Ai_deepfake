## рҹ§  About the Project

DeepScan is a **lightweight, web-based deepfake and AI-generated media verifier** built as an academic project. As synthetic media becomes increasingly indistinguishable from reality, tools that help everyday users verify digital content are critical.

DeepScan analyzes uploaded images using:
- **Pre-trained AI classification models** (CNN / EfficientNet-based)
- **Visual artifact heuristics** вҖ” texture inconsistencies, edge anomalies, frequency patterns
- **Metadata forensics** вҖ” EXIF data cross-referencing

The system outputs a **probability score** (e.g., `72% вҖ” Likely AI-Generated`) in a clean, user-friendly interface вҖ” no technical expertise required.

> вҡ пёҸ **Note:** This is a Light Version intended for academic and demonstration purposes. It does not provide legal or forensic-grade accuracy.

**Step-by-step:**

1. **Upload** вҖ” User submits an image or video frame via the web interface
2. **Preprocess** вҖ” Resize, normalize, strip and preserve EXIF metadata
3. **Artifact Analysis** вҖ” Heuristic scan for GAN fingerprints, texture anomalies, lighting inconsistencies
4. **Model Inference** вҖ” CNN classifier produces a real vs. synthetic logit score
5. **Metadata Forensics** вҖ” EXIF cross-referencing; missing/anomalous fields flagged as synthetic signals
6. **Score Fusion** вҖ” Weighted ensemble of model + heuristic + metadata scores
7. **Result Display** вҖ” Probability meter, verdict, and signal summary shown to user
